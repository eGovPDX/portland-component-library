# Button

<Subtitle>Action trigger for forms and interactions</Subtitle>

<Description>
The `Button` component draws attention to key actions with accessible markup and consistent styling. Use variants and sizes to match hierarchy and context.
</Description>

## Usage

```jsx
import { Button } from '@cityofportland/component-library/components/Button';
import '@cityofportland/component-library/components/Button/style.css';

export function Example() {
  return <Button variant="secondary">Submit</Button>;
}
```
